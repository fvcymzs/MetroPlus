📖项目介绍
===
MetroPlus是一款集多种功能的桌面端应用，旨在为用户提供武汉地铁的出行路线规划服务，并根据用户的偏好进行推荐。此外，MetroPlus还提供出行成本估算、实时地铁运营状况更新、个性化偏好设置、保存历史路线等服务。<br>

🚀功能特性
===
MetroPlus功能包括：<br>
  1.出行规划：用户输入起点和终点地铁站后，系统能够根据不同偏好（如站点最少、路程最短等）为用户规划地铁乘坐线路，并提供图形化展示。<br>
  2.实时地铁：根据历史班次，向用户提供发车时间间隔以及地铁线路信息。<br>
  3.保存路线：用户能够保存搜索过的路线，以便快速访问和重新规划。<br>
  4.运营时间：以武汉地铁官方为准，系统能够向用户提供运营时间表，包含不同线路、不同站点的首、末班车时间。<br>
  5.热门地点：系统能够向用户展示武汉的部分热门景点及商圈商场，并提供其一公里范围内的地铁站点信息。<br>
  6.个性主题：用户可以在皮肤界面切换系统预设的不同背景颜色及头像。<br>
